# Ilmcha

## Technologies Used

This project is primarily developed using:

- **TypeScript** (97.7%): Providing type safety and enhanced development experience.
- **JavaScript** (2.3%): For additional functionality and compatibility.

## Installation

Follow the steps below to set up the project locally and run:

1. Unzip the folder and open a terminal
2. Navigate to the project directory:
   ```bash
   cd Ilmcha
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage

- Using your mobile phone, install the Expo Go application, and scan the QR code
- Now you can try it out!
